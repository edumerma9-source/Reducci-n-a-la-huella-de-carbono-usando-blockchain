# 🌿 EcoIncentives Protocol

> **Innovating for a Sustainable Future Hackathon 2025**
> Transforming environmental conservation into a verifiable economic asset.

## 📖 Project Overview

EcoIncentives Protocol is a decentralized application (dApp) built on the **LACNet (EVM)** network. Our goal is to reduce the global carbon footprint by tokenizing ecological actions and aligning economic incentives through blockchain technology.

By providing a transparent, immutable ledger for carbon reductions, we eliminate the **information asymmetry** and high **transaction costs** present in traditional carbon credit markets.

## 🗂️ Repository Structure

This repository is organized to separate the smart contract logic from the user interface and project documentation:

* 📂 `contracts/`: Contains `EcoIncentives.sol`, our EVM-compatible smart contract.
* 📂 `Frontend/`: Contains `index.html` and assets for the web interface.
* 📂 `Presentation/`: Contains our `EcoIncentives_PitchDeck_EN GITHUB.pdf`.

## ⚙️ Technologies Used

* **Smart Contracts:** Solidity, EVM compatibility.
* **Blockchain Network:** LACNet (Open Pro-Testnet) to ensure zero fixed entry costs and sustainable network gas usage.
* **Frontend:** HTML/CSS/JavaScript (Web3 integration).

## 🚀 How to Run Locally

### 1. Smart Contract

The contract `EcoIncentives.sol` can be deployed using [Remix IDE](https://remix.ethereum.org/) or Hardhat/Truffle.

* Network: Custom RPC (LACNet)
* Compile using Solidity version `^0.8.0`

### 2. Frontend

No local server is strictly required for the basic UI.

1. Clone this repository: `git clone https://github.com/edumerma9-source/Reducci-n-a-la-huella-de-carbono-usando-blockchain.git`
2. Navigate to the `Frontend` folder.
3. Open `index.html` in your preferred web browser.
